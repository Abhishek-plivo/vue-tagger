# vue-tagger

[![Build Status](https://travis-ci.org/syropian/vue-tagger.svg?branch=master)](https://travis-ci.org/syropian/vue-tagger)

> Simple tagging component for Vue with built-in autocomplete.

## Install
```js
$ yarn add vue-tagger
```
_or_

```js
$ npm install vue-tagger --save
```
